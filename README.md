# ASOS-dataset-sample

<h2>1001件のASOS商品のサンプルdataset</h2>

![ASOS products dataset header](https://github.com/bright-jp/ASOS-dataset-sample/blob/main/ASOS-dataset.png)

1000件を超えるレコードを含むASOS商品のdatasetサンプルです。datasetは<b>Bright Data API</b>を使用して抽出されました。

<h2>ASOS商品のdatasetに含まれるデータポイントの一部:</h2>

* ```url```: Asos上の商品のリンク  
* ```name```: 商品名  
* ```brand```: 商品に関連付けられたブランド  
* ```description```: 商品を説明するテキスト詳細  
* ```about_me```: 商品に関する追加情報または詳細  
* ```availability```: 商品が現在利用可能かどうかを示します  
* ```buy_the_look```: ルック全体を購入するための情報  
* ```category```: 商品が属するカテゴリ  
* ```discount```: 商品に適用される割引  
* ```image```: 商品画像へのリンク  
* ```look_after_me```: 商品の手入れに関する説明または詳細  
* ```possible_sizes```: 商品の利用可能なサイズ  
* ```price```: 商品の価格  
* ```product_details```: 商品に関する追加の詳細  
* ```size_fit```: 商品のサイズとフィットに関する情報  
* ```you_might_also_like```: ユーザーが気に入る可能性のある他の商品候補  
* ```product_id```: 各商品の一意の識別子  
* ```breadcrumbs```: 商品の階層表現  
* ```currency```: 価格の表示通貨  
* ```color```: 商品で利用可能な色またはカラーバリエーション  

そのほかにも多数あります。

これは、<b>260万件超のレコード</b>を含む「ASOS products」datasetから派生したサンプルsubsetです。

利用可能なdatasetファイル形式: <b>JSON, NDJSON, JSON Lines, CSV, または Parquet。必要に応じて、ファイルは .gz に圧縮できます</b>。

dataset配信タイプのオプション: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>。

更新頻度: <b>1回限り、Daily、Weekly、Monthly、Quarterly、またはCustom basis</b>。

抽出されたデータポイントに追加できるデータエンリッチメント: <b>リクエストに応じて対応。</b>

<b>[Get the full ASOS products dataset](https://brightdata.jp/products/datasets/asos)</b>。

<h2>ASOS products datasetsのユースケースは何ですか?</h2>

<h3>1. 価格最適化</h3>
ASOS dataset内の類似するファッションアイテムやカテゴリを分析して競争力のある価格設定の機会を明らかにし、動的価格戦略を構築します。

<h3>2. 商品在庫</h3>
ASOS datasetを活用して在庫不足を特定し、特定のファッションアイテムに対する需要の高まりを検出することで、消費者に訴求するトレンドを明らかにします。

<h3>3. 市場戦略</h3>
ASOS datasetを活用して市場トレンドと消費者の嗜好を分析し、ファッション業界におけるより的確な戦略的意思決定を推進します。
