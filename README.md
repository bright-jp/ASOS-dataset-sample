# ASOS-dataset-sample

<h2>1001件のASOS商品のサンプルデータセット</h2>

![ASOS products dataset header](https://github.com/luminati-io/ASOS-dataset-sample/blob/main/ASOS-dataset.png)

1000件以上のレコードを含むASOS商品のデータセットサンプルです。データセットは <b>Bright Data API</b> を使用して抽出されています。

<h2>ASOS商品のデータセットに含まれるデータポイントの一部:</h2>

* ```url```: Asos上の商品のリンク  
* ```name```: 商品名  
* ```brand```: 商品に関連付けられたブランド  
* ```description```: 商品を説明するテキスト詳細  
* ```about_me```: 商品に関する追加情報または詳細  
* ```availability```: 商品が現在利用可能かどうかを示します  
* ```buy_the_look```: ルック全体を購入するための情報  
* ```category```: 商品が属するカテゴリ  
* ```discount```: 商品に適用される割引  
* ```image```: 商品画像へのリンク  
* ```look_after_me```: 商品のお手入れに関する指示または詳細  
* ```possible_sizes```: 商品の利用可能なサイズ  
* ```price```: 商品の価格  
* ```product_details```: 商品に関する追加詳細  
* ```size_fit```: サイズとフィットに関する情報  
* ```you_might_also_like```: ユーザーが気に入る可能性のある他の商品提案  
* ```product_id```: 各商品の一意識別子  
* ```breadcrumbs```: 商品の階層的な表現  
* ```currency```: 価格の通貨  
* ```color```: 商品の色、または利用可能なカラーオプション  

ほかにも多数あります。

これは「ASOS products」データセットから派生したサンプルサブセットであり、元のデータセットには <b>260万件以上のレコード</b> が含まれています。

利用可能なデータセットのファイル形式: <b>JSON, NDJSON, JSON Lines, CSV, or Parquet。必要に応じて、ファイルは .gz に圧縮できます</b>。

データセット配信タイプのオプション: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>。

更新頻度: <b>Once, Daily, Weekly, Monthly, Quarterly, or Custom basis</b>。

抽出されたデータポイントに追加できるデータエンリッチメント: <b>リクエストに基づきます。</b>

<b>[ASOS商品のフルデータセットを入手](https://brightdata.jp/products/datasets/asos)</b>。

<h2>ASOS商品のデータセットのユースケースは何ですか？</h2>

<h3>1. 価格最適化</h3>
ASOSデータセット内の類似ファッションアイテムやカテゴリを分析することで、競争上の価格機会を見つけ出し、動的な価格戦略を策定します。

<h3>2. 商品在庫</h3>
ASOSデータセットを活用して特定のファッションアイテムに対する在庫不足を特定し、需要の高まりを検知することで、消費者に訴求するトレンドを明らかにします。

<h3>3. 市場戦略</h3>
ASOSデータセットを活用して市場トレンドや消費者の嗜好を分析し、ファッション業界におけるより適切な戦略的意思決定を推進します。
